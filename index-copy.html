Course Syllabus

Module 1: Mathematics, Statistics and Programming for Data Science

+
  • Course Introduction and Overview
  • Programming Fundamentals for Data Science
  • Math & Statistics Fundamentals for Data Science
  • Data Science Practice (ongoing)

Module 2: Exploratory Data Analysis and Data Science Practices

+
  • Data Cleaning and Quality Assessment
  • Data Visualisation (Tableau)
  • Statistical Testing
  • Succeeding as a Data Scientist in Industry
  • Mini Project 1 – Data Science Practice

Module 3: Databases and APIs

+
  • Relational and Non-Relational Databases
  • SQL
  • Application Programming Interfaces (APIs)
  • Data Science Practice (ongoing)

Module 4: Machine Learning - Supervised Learning - Regression

+
  • Introduction to Machine Learning
  • Feature Modelling
  • Evaluating Regression Models
  • Overfitting and Regularisation

Module 5: Machine Learning - Supervised Learning - Classification

+
  • Logistic Regression
  • Evaluating Classification Models
  • Support Vector Machines
  • Bayesian Inference

Module 6: Machine Learning - Unsupervised Learning

+
  • K-means Clustering
  • Principal Components Analysis
  • Research and Documentation Training
  • Mini Project 2 – Data Science Practice

Module 7: Decision Trees and Ensemble Methods

+
  • Decision Trees
  • Random Forests
  • Bias and Variance
  • Bagging, Boosting, and Stacking

Module 8: Natural Language Processing and LLMs

+
  • Web Scraping
  • Text Pre-processing
  • Sentiment Analysis
  • Large Language Models

Module 9: Artificial Intelligence and Deep Learning

+
  • Introduction to Artificial Intelligence
  • Reinforcement Learning
  • Deep Feedforward Neural Networks
  • Recurrent & Convolutional Neural Networks

Module 10: Generative AI and Machine Learning Deployment

+
  • Generative AI Models
  • Deployment of Machine Learning Models
  • Cluster Computing
  • Streaming Data
  • Capstone Project

Frequently Asked Questions

What are the programme fees?

+

Speak to a career consultant about fee and fee support details. We offer various payment options including upfront payment and deposit plans.

What are the entry requirements?

+

Completion of a diploma in any field of studies, completion of any previous qualification undertaken in English or obtained IELTS 5.5, and completion of assigned online pre-work.

What is the time commitment for part-time?

+

24 weeks intensive remote training with 2 night classes per week plus 1 fortnightly Saturday class.

What is the time commitment for full-time?

+

12 weeks intensive remote training, Monday through Friday.

Where are classes held?

+

Classes are conducted remotely with options for in-person sessions at select locations.
